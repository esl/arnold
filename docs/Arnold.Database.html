    Arnold.Database (Arnold v0.6.2)

    API wrapper for Memento to store or fetch data from the database.

    Link to this section Summary

    Functions

    get(table, uuid)

    Fetches data from database. Needs a table schema and an uuid which is made from the combination of node_name and sensor_id with Arnold.Utilities.id/2.

    get(schema, node_id, sensor_id)

    Tries to fetch a specific node_id and sensor_id from a table. If not found returns all the currently saved data from the backend

    insert(data)

    Inserts the data to the specific table of the Postgres database. If it's there updates the value
