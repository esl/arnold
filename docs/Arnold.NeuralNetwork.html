    Arnold.NeuralNetwork (Arnold v0.6.2)

    The neural network API that contains function for training and prediction.

    Link to this section Summary

    Functions

    predict(sensor, tag, horizon, method)

    Predict a value or a list of values or a single value based on the method which can be :complex or :simple. Simple type utilizes the Single Exponential Smoothing algorithm from Arnold.Analyser.ExponentialSmoothing.simple/1, which is only good for predicting one value ahead. Complex uses neural network while also affects the results by trend and seasonality.

    save(neural_network_data)

    Saves the given result from the Arnold.NeuralNetwork.train/1 function.

    train(sensor)

    Trains the neural network with the given sensor and tag, until a configure eps value is not reached. If it fails to reach the given epsilon value, the training stops when exceeding the :max_iterates config value.

    predict(sensor, tag, horizon, method)

    (since 0.6.2) 
    @spec predict(
  sensor :: Arnold.Database.Table.Sensor.t() | nil,
  tag :: Arnold.Sensor.tag(),
  horizon :: pos_integer(),
  type :: prediction_method()
) :: {:ok, prediction :: prediction()}

    Predict a value or a list of values or a single value based on the method which can be :complex or :simple. Simple type utilizes the Single Exponential Smoothing algorithm from Arnold.Analyser.ExponentialSmoothing.simple/1, which is only good for predicting one value ahead. Complex uses neural network while also affects the results by trend and seasonality.

    If the sensor is nil it returns nil.
