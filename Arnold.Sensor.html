    put(node_id, sensor_id, timestamp, value)

    (since 0.6.0) 
    @spec put(
  node_id :: binary(),
  sensor_id :: binary(),
  time :: pos_integer(),
  value :: number()
) ::
  Arnold.Database.Table.Sensor.t()

    Puts a given sensor with a value into one of the agent's state. The function creates a new sensor if the agent failed to find it in its state or in the database, or if found one updates it.

    The agent calls the Arnold.Sensor.new/3 and Arnold.Sensor.update/3 functions.

    example

     Example 

    iex(1)> Arnold.Sensor.put("node", "sensor", 5)

    hourly: [{1610986020, 5}],
